一、 介绍

目前界面科室的取值类型有：全院科室、当前登录科室、关联科室、补货科室四种类型。由于项目个性化要求，部分界面科室取值不符合要求，可通过该页面进行配置修改，按安全组控制。

二、 操作说明

1. 查找菜单CSP链接:
说明:选中需维护的安全组，点击增加按钮，菜单代码列可通过菜单代码或菜单名称模糊检索,选中菜单时菜单名称及Csp链接会自动赋值

2. 查找菜单中科室控件的ID:
说明:Studio中打开CSP链接，找到需控制的科室控件ID，填写控件列

3. 维护菜单数据:
维护必填项后保存即可
科室类型说明：全部科室指当前登录院区的全部科室；补货科室指科室扩充信息维护界面维护的补货(接收)科室数据；关联科室值科室病区下维护的关联科室
