维护内容说明

【是否有效】：有效的传染病诊断会显示在传染病报告的诊断下拉框中，诊断录入界面符合传染病的诊断会有报卡弹窗；无效则不会在传染病报告的诊断中显示，同时诊断录入界面符合传染病的诊断也不会有报卡弹窗。
【多次患病】：勾选多次患病，在诊断录入时不管录入几次重复诊断都会提示或强制报卡；不勾选多次患病，只有第一次上报时会弹出报卡弹窗，后续再录入当前诊断则不会再弹出任何窗口，建议勾选！
【重复报告时限(天) 】：该传染病诊断第一次上报后再次诊断录入时，如果在重复报告时限天数之内（含结束日当天），录入该传染病诊断会提示报卡；如果超过重复上报时限，再次录入该传染病诊断则会强制报卡。多次患病为否时，无需维护该字段。
【是否强制报卡】：勾选强制报卡后，该诊断允许弹出强制报卡的诊断录入弹窗；不勾选强制报卡，该诊断的弹窗则一定为提示报卡弹窗；仅针对部分地区一直不强制上报的诊断，建议勾选！
