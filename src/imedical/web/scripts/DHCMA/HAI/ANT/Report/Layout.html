1.每一个使用抗菌药物为碳青霉烯类和替加环素类的患者均需上报。
2.范围为全部住院患者，重症医学科的患者需填写APACHEII评分。
3.前一次用药停药72小时后再次使用，即为二次用药，为1人次，2例次，其他以此类推。
4.患者标本送检时间如在首次开始用药时间前的5天之内细菌学培养送检及结果，记做本次用药的送检。送检标本应与感染部位相关。
5.微生物培养结果需填写纸片法K-B值（小于6记为<6，大于40记为>40，6-40之间需填写具体数值）或自动化法MIC值（小于0.004记为<0.004，大于512记为>512，0.004-512之间需填写具体数值）
