输入项目

【分类】：按医嘱所属类别选择是医嘱项、医嘱套或其他（建议关联准确的医嘱项，不建议关联医嘱套）。
【医嘱类型】：按医嘱项目优先级选择医嘱类型。
【医嘱名称】：可按医嘱汉字或首拼字母检索，默认从左到右匹配，使用”。“或”.“结尾时可进行全局匹配。
【关联号】：多个医嘱间存在关联关系时，可按1、1.1、1.2…维护关联关系，其中1为主医嘱，其他为从医嘱。
【用药天数】：长期类医嘱维护使用，默认为阶段天数。
【数量】：临时类医嘱维护使用。
【检查部位】:检查类医嘱维护，需对当前检查医嘱提前维护好适用部位。
【剂量】【剂量单位】【频次】【用法】【疗程】【数量单位】：根据医嘱名称选择后默认加载，可按需修改。

操作功能

【新增】输入医嘱相关项目后点击该按钮实现新增关联医嘱。
【修改】选中列表中某条已关联医嘱，并对记录相应内容编辑后点击该按钮，可实现对所选已关联医嘱的修改功能。
【删除】选中列表中某条已关联医嘱，点击该按钮，可实现对所选已关联医嘱的删除功能。
【复制】点击该按钮，在弹出的复制操作页面，左侧选择指定路径，右侧选中指定阶段及项目，可对该项目下已关联医嘱勾选并点击【保存】，勾选医嘱将自动关联到当前项目关联医嘱列表。
【上移】选中列表中某条已关联医嘱（非首行）点击该按钮，实现该记录向上移动一行功能。
【下移】选中列表中某条已关联医嘱（非尾行）点击该按钮，实现该记录向下移动一行功能。
【保存顺序】使用【上移】【下移】调整顺序后，需点击该按钮，才能保存调整后的顺序。
