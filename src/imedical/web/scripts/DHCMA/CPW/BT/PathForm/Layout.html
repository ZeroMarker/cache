    检查模式

    【诊断符合】：临床医生保存诊断时，匹配患者当日诊断是否符合准入规则，符合提示入径；
    【手术符合】：临床医生保存手术申请后，检查当此申请中拟施手术是否符合准入规则，符合提示入径；
    【诊断手术同时符合】：需将准入信息维护中”诊断手术组合“维护为是，诊断保存或手术申请保存后，检查当日诊断和最新申请中拟施手术是否同时符合准入规则，符合则提示入径；
    【中西医诊断同时符合】：需将准入信息维护中”中西诊断组合“维护为是，诊断保存时，需至少一个中医诊断、一个西医诊断同时符合才算诊断符合；
    【诊断匹配模式】：支持多种诊断匹配方式，a.诊断保存时只检查主诊断，不考虑诊断类型；b. 严格按照准入信息维护的诊断类型匹配临床诊断优先级；c.检查当日所开所有诊断，既不考虑是否主诊断，也不考虑诊断类型。以上三种诊断匹配模式可通过修改配置参数”CPWIsOnlyCheckMainDiag“实现。

    维护规则

    维护诊断/手术的代码和关键字时：
    若存在多个诊断或关键字，可使用英文逗号“,”进行分隔，符合该符号分隔的其一且不属于排除内容的即表示符合准入规则，英文逗号“,”表示“或者”的含义，如：a,b,c,d,e。
    在英文逗号分隔的内容中，如需同时满足多个条件，使用“&”进行连接，该符号表示“并且”的含义，如：a,b&c,d,e。
    诊断代码或手术代码维护时，支持维护到类目或亚目或细目均可，存在范围时，使用短横线“-”进行连接，短横线“-”可以表示范围，如：a&b,c-d,e。
    注意：规则维护中，不支持括号的使用。

    有排除的诊断和手术可先选择左侧某条规则记录，针对该记录添加需要排除的诊断和手术。
