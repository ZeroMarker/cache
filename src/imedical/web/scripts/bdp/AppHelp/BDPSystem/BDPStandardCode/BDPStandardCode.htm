国家标准编码调用说明

 

基础数据维护中整合了国家标准编码，先是在系统配置下开启国家标准编码功能，然后在系统配置下配置允许列表里可显示哪些字段，再在国家标准编码里进行对照，对照之后进入基础数据维护的页面，进行国家标准编码的维护。没有开启国家标准编码或者没有进行页面对照，则不在页面里显示国家标准标准数据。

 

一国家标准数据的配置

1.1国家/地区标准数据源类别

 

User表： User.BDPNationalDataType

表名称： BDP_NationalDataType

 

1.2 国家标准数据元值域

 

国家标准数据源值域里的类型引用了 国家标准数据类别的数据。

User表：User.BDPNationalDataDomain

表名称： BDP_NationalDataDomain

 

1.3系统配置-国家标准编码开启与字段显示

 

在系统配置下开启国家标准编码功能，开启后才能够使用国家标准数据，注意：在没对照之前页面也没法使用国家标准编码。

 

设置可以在列表里显示哪些国家标准编码的字段，设置后就可以在列表里显示要显示的字段。

 

 

1.4 国家/地区标准编码数据对照

 

国家/地区标准编码数据对照，用于对基础数据维护页面进行对照，对照了之后才可以在基础数据维护页面里使用国家标准数据。选中的对照项决定了在页面里可以使用哪些国家标准数据。

三 调用国家标准编码程序接口

 

用于数据保存的user 类： User.BDPStandardCode

表名： BDP_StandardCode

业务类： web.DHCBL.BDP.BDPStandardCode

3.1 查询某一个表的国家标准编码数据

方法名：GetStandardCode

入参： 表名 ,RowId

 

Return 格式：“国家标准编码”^“国家标准编码名称”^”医院标准编码“^”

医院标准编码名称”[上箭头间隔]

s resultStr=##class(web.DHCBL.BDP.BDPStandardCode).GetStandardCode(表名,RowId)

 

3.2使用范例

w ##class(web.DHCBL.BDP.BDPStandardCode).GetList("CT_Title","27") 

返回的数据为 　　DE01.00.002.00-1^初次报告^1^1
