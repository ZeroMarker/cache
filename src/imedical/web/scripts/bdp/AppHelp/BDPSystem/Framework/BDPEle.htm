功能元素授权

1.功能介绍

例如，用户界面中,未对功能元素授权时，界面中所有的操作都可用。(如图2-2-1)

图2-2-1

 

在【系统配置】->【功能元素授权】界面，选中安全组、科室或用户，如Demo Group用户，选中要授权的功能项，如“用户”，在功能元素授权模块，选择可编辑元素列表的元素，点击【→】或【→>】，点击【保存】按钮（图2-2-2）。再次进入用户界面，不可编辑元素所对应的操作不可用。（图2-2-3）

2.用法说明

注：如果调用方法时报错，请把基础数据平台更新到2.1.3以上版本

2.1调用方法

d ##class(web.DHCBL.BDP.BDPExecutables).BuildAutAry("表名")

 

调用此方法可以获取到该页面授权后元素属性值数组，返回类型为布尔类型，可以直接在JS中使用，同时支持ExtJs和Jquery等脚本语言。

 

 

蓝色区域为你的页面名代码(单表维护一版以表命名)，

 

2.2使用范例

2.2.1 ExtJs使用范例：

组件初始化时赋值

disabled : BDPAutDisableFlag('btnAdd')；

组件初始化完毕后赋值   

Ext.getCmp("form-save").getForm().findField('btnAdd').setDisabled(BDPAutDisableFlag('btnAdd'));

2.2.2 JQuery使用范例：

if(BDPAutDisableFlag('btnAdd')==true){$('#btnAdd').linkbutton('disable');}
