频次维护说明：

  • 1.ST（即刻） /ONCE（一次）的分发次数要求为1，间隔长度、间隔单位、周频次系数要求为空，且不允许勾选周频次，不需要维护分发时间。
  • 2.PRN（必要时）的分发次数要求为空，间隔长度、间隔单位、周频次系数要求为空，且不允许勾选周频次，不需要维护分发时间。
  • 3.【间隔长度】和【间隔单位】要成对维护。选了间隔长度、间隔单位，不允许勾选【不规则分发时间标志】。【间隔单位】为“小时”时，不需要维护分发时间。
  • 4.勾选了【不规则分发时间标志】配置后:
    （1）在分发时间页签添加数据时，需添加该分发时间对应的【分发次数描述】信息，描述必填。比如08:00:00对应早餐前，12:00:00对应午餐前等。正常频次修改为不规则分发时间频次后 ，需维护分发时间的描述信息。
    （2）【分发次数】为选填。若维护了分发次数，则业务上会限定该频次的分发时间个数。
    （3）【不规则分发时间标志】与【周频次】、【周频次系数】互斥。
    （4）【不规则分发时间标志】与【间隔长度】、【间隔单位】互斥。
  • 5.在没有勾选【不规则分发时间标志】且没有维护【间隔长度】、【间隔单位】为“小时”时，分发时间且分发时间要与【分发次数】对应（如Bid【分发次数】为2，就要维护两个分发时间），否则门诊结算时会报错、住院不会产生执行记录。
  • 6.【门诊/住院默认按分发次数全执行】：按若首日执行录入框可填，则默认为全部分发次数；若录入框不可填，则当日直接按照全部分发次数生成执行记录。
  • 7.对于中草药频次，中文描述为“饮片”。
